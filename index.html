Welcome to windIO’s documentation!¶

The repository defines two frameworks defining the inputs and outputs for systems engineering MDAO of wind turbine and plants. The github repository is here. The frameworks are implemented in two yaml-schemas, one for the turbine and one for the plant, to enforce an ontology of how data should be organized. The reference wind turbines designed within work package 2 of the IEA Wind Task 37 are checked against the schema through unit testing. You can access the yaml files at the following links IEA onshore 3.4 MW, IEA offshore 10.0 MW, and IEA floating 15.0 MW.
